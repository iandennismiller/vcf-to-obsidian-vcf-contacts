vcf-to-obsidian-vcf-contacts documentation
==========================================

This is the API documentation for vcf-to-obsidian-vcf-contacts, a Python module that 
batch-converts VCF files into Markdown files that are compatible with the 
`obsidian-vcf-contacts <https://github.com/broekema41/obsidian-vcf-contacts>`__ plugin for ObsidianMD.

Overview
--------

vcf-to-obsidian-vcf-contacts provides a comprehensive solution for converting VCF 
(vCard) contact files into Markdown format suitable for use in Obsidian with the 
vcf-contacts plugin. The library includes:

* VCF file reading and parsing
* Markdown content generation with frontmatter
* Intelligent filename generation
* Command-line interface
* Batch processing capabilities
